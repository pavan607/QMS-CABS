# Quality Management System - User Guide

## Overview

The Quality Management System (QMS) is a comprehensive web application designed to help organizations manage their quality processes, documents, inspections, and compliance activities.

## Getting Started

### Accessing the System

1. Open your web browser
2. Navigate to the QMS URL (default: http://localhost:3000)
3. You will be redirected to the login page

### Logging In

1. Enter your email address
2. Enter your password
3. Click "Sign In"

**Default Admin Credentials** (for initial setup):
- Email: admin@qms.com
- Password: admin123

⚠️ **Important**: Change the default password immediately after first login.

## User Interface

### Header

The header appears at the top of every page and contains:

- **Menu Toggle Button** (left): Click to show/hide the sidebar
- **System Name** (center-left): "Quality Management System"
- **User Profile** (right): Shows your name, email, and provides access to sign out

### Sidebar Navigation

The left sidebar provides access to all major sections:

- **Dashboard**: Overview and statistics
- **Documents**: Manage quality documents
- **Quality Checks**: Track inspections and audits
- **Reports**: View and generate reports
- **Users**: Manage system users (admin only)
- **Settings**: Configure system settings

The sidebar can be collapsed or expanded using the menu button in the header.

### Footer

The footer appears at the bottom of every page with:
- Copyright information
- Links to Privacy Policy, Terms of Service, and Support

## Main Features

### 1. Dashboard

The dashboard provides an at-a-glance view of your quality management activities:

**Statistics Cards:**
- Total Documents
- Quality Checks performed
- Active Users
- Generated Reports

**Recent Activity:**
- Recently updated documents
- Latest quality check results
- Quick access to view details

### 2. Documents Management

**View Documents:**
- Browse all quality documents in a table format
- See document name, type, date, and status
- Use the search bar to find specific documents

**Add New Document:**
1. Click the "Add Document" button
2. Fill in document details
3. Upload the file
4. Save

**Document Actions:**
- **View**: Opens the document for reading
- **Edit**: Modify document details

### 3. Quality Checks

**Overview:**
- View total checks, passed, and failed statistics
- Browse all quality inspections and audits

**Check Details:**
- Inspector name
- Check date
- Result (Passed/Failed)
- View detailed check reports

**Create New Check:**
1. Click "New Check"
2. Enter check details
3. Assign inspector
4. Save

### 4. Reports

**Available Reports:**
- Monthly Quality Summary
- Audit Compliance Report
- Inspection Statistics
- Training Completion Report

**Actions:**
- **View Report**: Opens the report in a new view
- **Download**: Downloads the report as PDF

### 5. User Management (Admin Only)

**View Users:**
- See all system users
- View their roles and status

**Add New User:**
1. Click "Add User"
2. Enter user details (name, email)
3. Assign role (Admin, Manager, Inspector, User)
4. Set initial password
5. Save

**User Roles:**
- **Admin**: Full system access
- **Manager**: Manage documents and reports
- **Inspector**: Perform quality checks
- **User**: View-only access

**User Actions:**
- **Edit**: Modify user details and permissions
- **Delete**: Remove user from system

### 6. Settings

**General Settings:**
- Organization Name
- Time Zone configuration
- System preferences

**Notification Settings:**
- Email Notifications toggle
- Quality Alerts configuration
- Audit Reminders

## Best Practices

### Document Management

1. **Naming Convention**: Use clear, descriptive names for documents
2. **Version Control**: Update documents regularly and track changes
3. **Status Management**: Keep document status up-to-date (Active, Review, Archived)

### Quality Checks

1. **Regular Inspections**: Schedule periodic quality checks
2. **Documentation**: Always document findings and corrective actions
3. **Follow-up**: Track failed checks and verify corrective measures

### User Access

1. **Principle of Least Privilege**: Assign minimum necessary permissions
2. **Regular Review**: Periodically review user access and roles
3. **Deactivate**: Disable accounts for users who no longer need access

## Security

### Password Requirements

- Minimum 6 characters
- Change default passwords immediately
- Use strong, unique passwords

### Session Management

- Sessions expire after period of inactivity
- Always sign out when finished
- Don't share login credentials

## Troubleshooting

### Cannot Log In

**Check:**
- Email address is correct
- Password is correct
- Caps Lock is not on
- Account is active

**Solution:**
- Contact administrator to reset password
- Verify account status

### Cannot Access Certain Pages

**Reason:** Insufficient permissions

**Solution:**
- Contact administrator to request access
- Verify your user role

### Page Not Loading

**Try:**
- Refresh the page (F5 or Ctrl+R)
- Clear browser cache
- Try a different browser
- Check internet connection

## Keyboard Shortcuts

## Glossary

- **QMS**: Quality Management System
- **Audit**: Systematic examination of quality processes
- **Inspection**: Detailed check of products or processes
- **Compliance**: Adherence to quality standards and regulations
- **Document Control**: Managing creation, review, and approval of documents
